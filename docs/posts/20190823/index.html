Mac 将 App 程序打包成为 dmg

2019/08/23 #code

用最简单的打包方式，将自己开发的App打包成为DMG，实现共享分发，快速安装

1. 新建DMG

打开磁盘工具，新建DMG

File->New Image->Blank Image

创建DMG

2. 添加文件

在磁盘工上右键打开

右键->Show in Finder

创建快捷键

ln -s /Applications Applications

完成文件添加

3. 排版，添加背景

修改排版

右键->Show View Options

自定义背景

4. 压缩DMG

推出DMG

转换DMG

压缩DMG

压缩后的文件明显小于原文件，也不可再进行编辑

5. 完成打包

至此，完成了App打包DMG，可以进行共享发布了
