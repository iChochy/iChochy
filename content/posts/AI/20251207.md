## 通用协议调用
直接调用 ZenMux API
```bash
curl https://zenmux.ai/api/v1/chat/completions
  -H "Content-Type: application/json"
  -H "Authorization: Bearer $ZENMUX_API_KEY"
  -d '{
    "model": "openai/gpt-5",
    "messages": [
      {
        "role": "user",
        "content": "了解回忆中的明天(https://ichochy.com）？"
      }
    ]
  }'

```
